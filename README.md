# Crypto Scraper

This Python script scrapes cryptocurrency data from [CoinMarketCap](https://coinmarketcap.com) including rank, name, symbol, and price.  

**Educational purpose only.**

## Requirements
- Python 3.x
- requests
- BeautifulSoup4 (`pip install beautifulsoup4 requests`)

## How to run
```bash
python crypto_scraper.py
